# Weather App - Homework Project

This is a simple weather dashboard created as part of a homework assignment. It provides a clean interface to search for weather information and view weekly forecasts.

## 👤 Project Details
- **Author:** Sahil Khan
- **Project Type:** Frontend Homework
- **Technologies:** HTML5, CSS3

## 🛠️ Features
- Responsive design for city search.
- Clean and modern UI with card-based layouts.
- Weekly weather forecast table.

## 📂 Project Structure
- `index.html`: Main structure of the weather app.
- `style.css`: Custom styling for the dashboard.
